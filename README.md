# BrickGame
##Emulator of retro device BrickGame

It requires SDL2 and SDL2_ttf installed

If font not working, download *Arial Narrow Bold* font, and put it to ./res/ARIALNB.TTF

Compile:
```sh
 $ make
```
Run emulator:
 ```sh
 $ cd build
 $ ./BrickGame
 ```
 Delete executable file:
 ```sh
 $ make clean
 ```
